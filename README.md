# Natsuki

夏希是一個基於 PHP 的時間類別，用以取得現在的時間，或者是數天後的現在時刻，

你也可以將時間轉換為相對時間（例如 2015 年 -> 1 年前），你也可以取得是否為休假日的布林值，

在這個時間類別裡，你可以到達不是現在的那一刻。
